# Objects

Methods to create objects
- Object Literal syntax 
`js
 const circle = {
 radius:1,
 location:{
  x:1,
  y:1
 },
 draw: function(){
  console.log("draw")
 }
}; //Object Literal syntax
// Has 3 members radius, Location, Draw.
// Properties are radius and location
// While draw is a property. 
`
